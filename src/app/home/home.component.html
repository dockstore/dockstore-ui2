Remote Tools and Workflows

Don't worry about migrating your documents! Dockstore references docker images hosted on Quay.io and DockerHub, allowing for quick setup of existing docker images. Descriptor files can be retrieved from GitHub, BitBucket and GitLab.

Hosted Tools and Workflows

Have your docker image registered but don't want to use source control? Store your workflow descriptor files directly on Dockstore. An in browser editor with syntax highlighting and basic validation is available.

Organisations and Collections

Create organisation landing pages for your company, start-up, working group, or university where you can create collections of related tools and workflows to share.
